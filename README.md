# ParamIneq — Неравенства с параметрами

Интерактивная образовательная веб-платформа для старшеклассников (ЕНТ, олимпиады). Помогает связать аналитическое решение неравенств с параметром и графическую визуализацию.

## Стек

- HTML5, CSS3 (glassmorphism, тёмная тема)
- Модульный JavaScript (ES modules)
- Canvas API — графики и подсветка областей решения
- MathJax 3 — формулы

## Запуск

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
npx serve .
# или
python -m http.server 8080
```

> ES-модули требуют HTTP(S); при открытии файла напрямую (`file://`) используйте локальный сервер.

## Разделы

1. **Главная** — hero с анимированной сеткой
2. **Теория** — карточки с формулами
3. **Визуализация** — ползунок параметра `a`, график, пошаговое решение
4. **База задач** — 8 примеров (§1.2, примеры 1–8)
5. **Практика** — тесты и прогресс в `localStorage`
6. **О проекте** — исследование и сравнение с GeoGebra

## Содержание учебника

Теория §1.1–1.2 и примеры 1–8 в `js/data/theory.js` и `js/data/tasks.js` соответствуют материалу «Неравенства с параметром» (классификация, линейные и квадратные неравенства, модуль, ОДЗ).
